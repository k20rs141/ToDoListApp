
# SwiftUI-ToDoList App


## 概要

`SwiftUI`と`CoreData`を使用したシンプルなToDoアプリです。

新規タスクの登録、削除まで行えます。また、ダークモードにも対応しています。


## 開発環境

- Xcode Version 14.2 or later
- iOS 15.0 or later 
- SwiftUI 4.0
